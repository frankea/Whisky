   - `displayName: String` (e.g. "Visual C++ Runtime")

  <action>
Create `Whisky/Utils/DependencyManager.swift` with GPL v3 header:

1. Define `DependencyManager` as a caseless enum (following GPUDetection/GameMatcher pattern) for static utility methods.
2. `static func checkDependencies(for bottle: Bottle, definitions: [DependencyDefinition] = DependencyDefinition.standardDependencies) async -> [DependencyStatus]`:
   - Call `Winetricks.loadInstalledVerbs(for: bottle)` (existing method from Phase 2)
   - For each definition, check if ALL its winetricksVerbs are in the installed set
   - Return DependencyStatus with `.installed`/`.notInstalled`/`.partiallyInstalled` and confidence based on whether data was from cache or fresh
3. `static func recommendedDependencies(for program: Program, bottle: Bottle) async -> [DependencyDefinition]`:
   - Check if program is ClickOnce -> recommend dotnet48
   - Check crash diagnosis history for DLL-not-found patterns -> recommend matching dependency
   - Check game config database for required verbs -> cross-reference with standardDependencies
   - Return only high-confidence recommendations (per user decision: never recommend without evidence)
4. `static func dismissRecommendation(_ definitionId: String, for program: Program)`:
   - Track dismissed recommendations in program settings (simple Set<String> stored on ProgramSettings or sidecar)
   - Don't reappear until new evidence
5. Add logger with category "DependencyManager".

Create `Whisky/Utils/Winetricks+Install.swift` with GPL v3 header:

1. Extension on `Winetricks` adding headless verb installation.
2. Define `WinetricksInstallProgress: Sendable` enum: `.preparing`, `.output(String)`, `.completed(exitCode: Int32)`, `.failed(String)`.
3. `static func installVerb(_ verb: String, for bottle: Bottle, timeout: TimeInterval = 600) -> AsyncStream<WinetricksInstallProgress>`:
   - Build environment matching `listInstalledVerbs` (WINEPREFIX, WINE, PATH, HOME)
   - Create Process with arguments: ["bash", winetricksURL.path, verb]
   - Pipe stdout and stderr, stream lines as `.output(String)`
   - Use same timeout pattern as listInstalledVerbs (Task.sleep + terminate)
   - On termination, yield `.completed(exitCode:)` or `.failed(message)`
   - After completion, refresh WinetricksVerbCache via `loadInstalledVerbs`
4. `static func installVerbs(_ verbs: [String], for bottle: Bottle) -> AsyncStream<(verb: String, progress: WinetricksInstallProgress)>`:
   - Sequential verb installation (queued plan per user decision)
   - Yields per-verb progress tuples
   - Continues on partial failure (later verbs still attempted)

This follows the existing `listInstalledVerbs` headless Process pattern exactly (not Terminal AppleScript), per user decision to attribute volume access to Whisky.
