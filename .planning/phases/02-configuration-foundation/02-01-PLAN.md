  <behavior>
    ## EnvironmentBuilder

    Layer enum with 8 cases ordered by rawValue (0-7):
      base(0), platform(1), bottleManaged(2), launcherManaged(3), bottleUser(4), programUser(5), featureRuntime(6), callsiteOverride(7)

    Builder resolution:
      - set("KEY", "a", .base) then set("KEY", "b", .platform) -> resolve().environment["KEY"] == "b" (later layer wins)
      - set("A", "1", .base) and set("B", "2", .platform) -> both keys present in resolved env
      - remove("KEY", .bottleManaged) after set("KEY", "val", .base) -> KEY absent in resolved env
      - setAll(["X": "1", "Y": "2"], .bottleUser) -> both keys set in that layer
      - Empty builder -> resolve() returns empty dict and empty provenance

    Provenance tracking:
      - resolve().provenance.entries["KEY"].layer == winning layer
      - resolve().provenance.entries["KEY"].overriddenBy == higher layer if overridden, nil if winning
      - resolve().provenance.activeLayers == set of layers that contributed at least one entry

    ## DLL Override Resolution

    DLLOverrideMode cases: builtin("b"), native("n"), nativeThenBuiltin("n,b"), builtinThenNative("b,n"), disabled("")

    DLLOverrideEntry: dllName (String, no .dll suffix) + mode (DLLOverrideMode), Codable + Equatable

    DLLOverrideResolver resolution (per-DLL, most specific wins):
      - managed=[dxgi=n,b, d3d11=n,b], bottleCustom=[], programCustom=[] -> "d3d11=n,b;dxgi=n,b" (sorted)
      - managed=[dxgi=n,b], bottleCustom=[vcrun=n], programCustom=[] -> "dxgi=n,b;vcrun=n"
      - managed=[dxgi=n,b], bottleCustom=[dxgi=b], programCustom=[] -> "dxgi=b" (bottleCustom wins)
      - managed=[dxgi=n,b], bottleCustom=[], programCustom=[dxgi=disabled] -> "dxgi=" (programCustom wins)
      - All empty -> empty string (no WINEDLLOVERRIDES)

    DLL override warnings:
      - managed=[dxgi=n,b] (source=.dxvk), bottleCustom=[dxgi=b] -> warning for dxgi: "overrides DXVK"
      - managed=[dxgi=n,b] (source=.dxvk), programCustom=[dxgi=native] -> warning for dxgi: "overrides DXVK"
      - No DXVK-managed DLLs overridden -> no warnings

    DLL override presets:
      - DXVK preset: [dxgi=n,b, d3d9=n,b, d3d10core=n,b, d3d11=n,b]
      - Presets return [DLLOverrideEntry] (transparent, editable entries)

    ## ProgramOverrides

    All fields Optional (nil = inherit from bottle):
      - dxvk: Bool?, dxvkAsync: Bool?, dxvkHud: DXVKHUD?
      - enhancedSync: EnhancedSync?
      - forceD3D11: Bool?
      - performancePreset: PerformancePreset?, shaderCacheEnabled: Bool?
      - controllerCompatibilityMode: Bool?, disableHIDAPI: Bool?, allowBackgroundEvents: Bool?, disableControllerMapping: Bool?
      - dllOverrides: [DLLOverrideEntry]?

    Codable behavior:
      - ProgramOverrides() (all nil) encodes to plist with no keys, decodes back as all nil
      - ProgramOverrides(dxvk: true) round-trips correctly; other fields remain nil
      - Plist missing "overrides" key decodes ProgramSettings with overrides == nil
      - isEmpty returns true when all fields nil, false otherwise

    ## Data model extensions

    ProgramSettings gains: var overrides: ProgramOverrides? (default nil, decodeIfPresent)
    BottleSettings gains: var customDLLOverrides: [DLLOverrideEntry] (default [], decodeIfPresent, public proxy property)
