**GameMatcher.swift:**
Create `GameMatcher` as a caseless enum (following GPUDetection pattern):

**Public types:**
`ProgramMetadata` struct (Sendable): exeName (String), exeURL (URL?), fileSize (Int64?), peTimestamp (Date?), steamAppId (Int?), installPath (String?)

**Public API:**
- `static func match(metadata: ProgramMetadata, against entries: [GameDBEntry]) -> [MatchResult]` -- score all entries against the metadata, return sorted by confidence descending, filtered to >= 0.3 threshold
- `static func bestMatch(metadata: ProgramMetadata, against entries: [GameDBEntry]) -> MatchResult?` -- return top result if confidence >= 0.7 AND (no close second match within 0.1 gap OR top is hardIdentifier tier)
- `static func searchEntries(_ query: String, in entries: [GameDBEntry]) -> [GameDBEntry]` -- filter entries by tokenized name match against title + aliases. Case-insensitive. Return entries where any token matches.

**Private scoring methods:**
- `scoreHardIdentifiers(metadata:, entry:) -> (Double, String)?` -- check Steam App ID match (1.0 / 0.98 for appmanifest vs steam_appid.txt), SHA256 match (0.99), file size + PE timestamp match (0.95). Return highest score with explanation string.
- `scoreStrongHeuristics(metadata:, entry:) -> (Double, String)?` -- check exact exe name match (0.85, or 0.90 if path pattern also matches). Penalize generic exe names (-0.3). Return score + explanation.
- `scoreFuzzy(metadata:, entry:) -> (Double, String)?` -- tokenize exe name (split on dots, dashes, underscores, spaces; remove .exe), compare tokens against title + aliases tokens. Score = matched tokens / total query tokens, scaled to 0.3-0.6 range. Add +0.1 bonus if install path contains a path pattern.

**Generic exe denylist:**
```swift
private static let genericExeNames: Set<String> = [
    "launcher.exe", "setup.exe", "start.exe", "game.exe",
    "app.exe", "install.exe", "uninstall.exe", "updater.exe",
    "config.exe", "settings.exe", "crash_reporter.exe",
    "ue4-win64-shipping.exe", "unity.exe", "unitycrashhandler64.exe"
]
```

**Variant auto-selection:**
For each MatchResult, select the recommended variant by:
1. Filter variants whose constraints match current machine (check CPU arch via ProcessInfo.processInfo.machineArchitecture, macOS version via ProcessInfo.processInfo.operatingSystemVersion)
2. If multiple match, prefer the one with `isDefault == true`
3. Fall back to first variant
