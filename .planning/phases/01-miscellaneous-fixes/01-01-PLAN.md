  <name>Task 2: Refactor ClipboardManager to return ClipboardCheckResult and add BottleCleanupConfig</name>
  <files>

  <action>
  **ClipboardManager refactoring:**

  The current `sanitizeForMultiplayer()` directly shows an NSAlert, which couples WhiskyKit to UI presentation. Refactor to a result-returning pattern:

  1. Add a public enum `ClipboardCheckResult` to ClipboardManager.swift:
     ```swift
     public enum ClipboardCheckResult: Sendable {
         /// Content is small or empty, no action needed
         case safe
         /// Content was auto-cleared (for known multiplayer launchers)
         case autoCleared(contentType: String, sizeBytes: Int)
         /// Content is large and needs user decision (show alert/sheet in app layer)
         case needsUserDecision(contentType: String, sizeBytes: Int, textPreview: String?)
     }
     ```
     Where `contentType` is "text", "image", or "other".

  2. Replace `sanitizeForMultiplayer()` with a new `checkBeforeLaunch(launcher:policy:threshold:) -> ClipboardCheckResult` method:
     - `launcher: LauncherType?` -- detected launcher (optional)
     - `policy: ClipboardPolicy` -- the per-bottle clipboard policy enum
     - `threshold: Int` -- the size threshold in bytes (default `Self.largeContentThreshold`)
     - Returns `.safe` if content is below threshold or policy is `.never`
     - Returns `.autoCleared(...)` if policy is `.alwaysClear` or (policy is `.auto` and launcher?.usesClipboard == true), after calling `clear()`
     - Returns `.needsUserDecision(...)` if policy is `.auto` (no launcher match) or `.alwaysWarn`
     - The `textPreview` is the first 50 characters of text content, nil for non-text

  3. Remove the private `showLargeClipboardAlert()` method entirely (alert presentation moves to app layer in Plan 03).

  4. Add a public enum `ClipboardPolicy: String, Codable, CaseIterable, Sendable`:
     ```swift
     case auto = "auto"           // Auto-clear for known launchers, warn for others
     case alwaysWarn = "warn"     // Always show warning
     case alwaysClear = "clear"   // Always clear on launch
     case never = "never"         // Never warn or clear
     ```

  **BottleCleanupConfig (new file):**

  Create `WhiskyKit/Sources/WhiskyKit/Whisky/BottleCleanupConfig.swift` with GPL header for WhiskyKit target:
  - `public struct BottleCleanupConfig: Codable, Equatable`
  - Properties:
    - `var clipboardPolicy: ClipboardPolicy = .auto`
    - `var clipboardThreshold: Int = ClipboardManager.largeContentThreshold`
    - `var killOnQuit: KillOnQuitPolicy = .inherit`
  - Add `public enum KillOnQuitPolicy: String, Codable, CaseIterable, Sendable`:
    - `case inherit = "inherit"` (use global killOnTerminate)
    - `case alwaysKill = "always"` (always kill on quit)
    - `case neverKill = "never"` (never kill on quit)
  - Include a defensive `init(from decoder:)` with decodeIfPresent defaults for all properties.

  **BottleSettings integration:**

     - `public var clipboardThreshold: Int { get/set }`
     - `public var killOnQuit: KillOnQuitPolicy { get/set }`
  3. Follow the existing pattern of other config sections (wineConfig, metalConfig, etc.)
