  <action>
Create directory `Whisky/Views/Install/` if it doesn't exist. Create `Whisky/Views/Install/DependencyInstallSheet.swift` with GPL v3 header:

1. Define `DependencyInstallSheet: View` with:
   - `let definition: DependencyDefinition`
   - `@ObservedObject var bottle: Bottle`
   - `@Environment(\.dismiss) private var dismiss`
   - `@State private var stage: InstallStage = .info`
   - `@State private var logLines: [String] = []`
   - `@State private var isInstalling: Bool = false`
   - `@State private var installResult: InstallResult?`
   - `@State private var showLog: Bool = false`

2. Define `InstallStage` enum: `.info` (What), `.preflight` (checks), `.plan` (verb list), `.running` (progress), `.verify` (re-check).

3. The sheet presents a staged flow (per user decision: guided install sheet with 5 sections):

   **Stage .info** ("What you're installing"):
   - Definition displayName as title
   - Description as body text
   - "This will install: {verbs joined}" as detail
   - Estimated time: "{estimatedInstallMinutes} minutes"
   - "Continue" button -> .preflight

   **Stage .preflight** ("Preflight checks"):
   - Check prefix health (WinePrefixValidation.validatePrefix)
   - Show results: green check for each passing, red x for failing
   - If all pass: "Install" button -> .running
   - If failing: show warning with option to proceed anyway

   **Stage .plan** (embedded in preflight):
   - Show diff-style verb list with verb names
   - Irreversibility warning: "Installed components cannot be automatically removed"

   **Stage .running** ("Installing..."):
   - ProgressView with indeterminate progress
   - Collapsible live log (DisclosureGroup, collapsed by default):
     * ScrollView with monospace text showing logLines
     * Auto-scroll to bottom
   - Stream output from `Winetricks.installVerb()` (from Plan 03)
   - On each `.output(line)`: append to logLines
   - On `.completed(exitCode)`: transition to .verify

   **Stage .verify** ("Verification"):
   - Re-run `DependencyManager.checkDependencies` for just this definition
   - Show result: "Installation successful" (green) or "Installation may have failed" (orange)
   - "Done" button dismisses sheet
   - Save install attempt to BottleDependencyHistory

4. Sheet frame: minWidth 500, minHeight 400. Use VStack with stage-based @ViewBuilder.

5. Cancel button available at all stages (pre-running). During .running, Cancel terminates the process.
