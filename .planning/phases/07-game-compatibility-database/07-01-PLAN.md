**GameDBRating.swift:**
Create `CompatibilityRating` enum at module scope (following ClipboardPolicy/GraphicsBackend pattern):
- Cases: `works`, `playable`, `unverified`, `broken`, `notSupported`
- Conform to `String, Codable, Sendable, CaseIterable, Comparable`
- Raw values: `"works"`, `"playable"`, `"unverified"`, `"broken"`, `"notSupported"`
- Add `displayName: String` computed property returning localized user-facing names
- Add `systemImage: String` computed property returning appropriate SF Symbol names (checkmark.circle, exclamationmark.triangle, questionmark.circle, xmark.circle, nosign)
- Implement Comparable based on severity ordering (works < playable < unverified < broken < notSupported)

**GameDBEntry.swift:**
Create the following types, all `Codable, Sendable, Equatable`:

`ExeFingerprint`: sha256 (String?), fileSize (Int64?), peTimestamp (Date?)

`GameConstraints`: cpuArchitectures ([String]?), minMacOSVersion (String?), minWineVersion (String?), requiredBackendCapabilities ([String]?)

`TestedWith`: lastTestedAt (Date), macOSVersion (String), wineVersion (String), whiskyVersion (String?), cpuArchitecture (String?)

`KnownIssue`: description (String), severity (String?), workaround (String?)

`Provenance`: source (String), author (String?), lastUpdated (Date?), referenceURL (URL?)

`GameConfigVariantSettings`: Map directly to BottleSettings property names:
  - graphicsBackend (GraphicsBackend?), dxvk (Bool?), dxvkAsync (Bool?), enhancedSync (EnhancedSync?), forceD3D11 (Bool?), performancePreset (String?), shaderCacheEnabled (Bool?), avxEnabled (Bool?), sequoiaCompatMode (Bool?)
  - All optional with decodeIfPresent defaults (following BottleSettings defensive pattern)

`GameConfigVariant`: id (String), label (String), isDefault (Bool?), whenToUse (String?), rationale ([String]?), settings (GameConfigVariantSettings), environmentVariables ([String: String]?), dllOverrides ([DLLOverrideEntry]?), winetricksVerbs ([String]?), testedWith (TestedWith?)

`GameDBEntry`: id (String), title (String), aliases ([String]?), subtitle (String?), store (String?), steamAppId (Int?), rating (CompatibilityRating), exeNames ([String]?), exeFingerprints ([ExeFingerprint]?), pathPatterns ([String]?), antiCheat (String?), constraints (GameConstraints?), variants ([GameConfigVariant]), notes ([String]?), knownIssues ([KnownIssue]?), provenance (Provenance?)

Use defensive `decodeIfPresent` with safe defaults for all optional fields in custom `init(from:)`. Add `defaultVariant: GameConfigVariant?` computed property returning first variant where `isDefault == true`, falling back to first variant.

**MatchResult.swift:**
`MatchTier` enum: hardIdentifier (rawValue 3), strongHeuristic (2), fuzzy (1). Conform to Int, Codable, Sendable, Comparable.

`MatchResult` struct (Sendable): entry (GameDBEntry), confidence (Double, 0.0...1.0), tier (MatchTier), explanation (String), recommendedVariant (GameConfigVariant?)

**SteamAppManifest.swift:**
`SteamAppManifest` caseless enum (following PatternLoader/GPUDetection pattern):
- `static func parseAppId(from text: String) -> Int?` -- parse Valve ACF/VDF format to extract `"appid"` value. ACF uses `"key" "value"` pairs with optional whitespace and tab indentation.
- `static func findAppId(in bottleURL: URL) -> Int?` -- scan `drive_c/Program Files (x86)/Steam/steamapps/` for `appmanifest_*.acf` files, parse each, return first match. Also check for `steam_appid.txt` in common locations.
- `static func findAppIdForProgram(at exeURL: URL) -> Int?` -- check for `steam_appid.txt` in exe directory and parent directories (up 3 levels). Parse as plain integer text.
